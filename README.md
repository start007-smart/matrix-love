## ✨ 特效介绍

这个项目包含多层炫酷特效：

### 1. **矩阵代码雨背景**
- 经典的黑客帝国绿色代码雨
- 包含数字、字母和日文片假名字符
- 营造赛博朋克氛围

- 使用CSS 3D变换实现的立体爱心

## 🎨 技术特点

- **纯原生技术**：HTML5 + CSS3 + JavaScript，无需任何框架
- **Canvas动画**：使用Canvas API实现复杂粒子系统
- **数学之美**：使用参数方程绘制完美爱心曲线
- **性能优化**：使用requestAnimationFrame确保流畅动画
- **响应式设计**：支持不同屏幕尺寸

## 💡 自定义建议

你可以轻松修改以下内容：

1. **修改文字内容**：
   - 编辑 `script.js` 中的 `typingTexts` 数组
   - 修改 `index.html` 中的标题文字

2. **调整颜色**：
   - 主色调：`#00ff41`（矩阵绿）
   - 辅助色：`#00ff88`（青绿色）
   - 可在 `style.css` 中全局替换

3. **调整动画速度**：
   - 修改各个类中的 `speed` 参数
   - 调整CSS动画的 `duration`
